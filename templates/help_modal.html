Search Field Instructions

The search function has five basic features, demonstrated by these examples:


First Name Last Name Country Capacity ...
Albert Smith USA 500 ...
Farid Sidi Algeria 400 ...
Benjamin Huber Germany 200 ...
Alberto Benson Ireland 200 ...
... ... ... ... ...

The database is searched for entries that (partially) include al or ben in any fields.


First Name Last Name Country Capacity ...
Albert Smith USA 500 ...
Benjamin Huber Germany 200 ...
... ... ... ... ...

This is the same search but only the specified countries are shown.


First Name Last Name Country Capacity ...
Albert Smith USA 500 ...
Farid Sidi Algeria 400 ...
... ... ... ... ...

For fields with numbers, searches can be filtered using the greater-than or less-than operators (> and <).


First Name Last Name Country Capacity ...
Albert Smith USA 500 ...
... ... ... ... ...

Double quotes can be used to match entries exactly (In this example only Albert is shown, but Alberto is not).


First Name Last Name Country Capacity ...
Albert Smith USA 500 ...
... ... ... ... ...

Use the asterisk symbol (*) to search for a specific full name.
